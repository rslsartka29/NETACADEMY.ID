Tentang Media

💻 Latar Belakang Aplikasi

NETACADEMY merupakan media pembelajaran berbasis web responsive yang dikembangkan sebagai sumber belajar mandiri pada materi Jaringan Komputer dan Internet. Media ini menerapkan prinsip Infografis Interaktif untuk mempermudah visualisasi konsep abstrak jaringan agar lebih konkret dan menarik bagi siswa Kelas X.

🛠️ Spesifikasi Teknis
Metode Validasi Ahli Media & Ahli Materi
Teknologi Utama HTML5, CSS3, JavaScript (ES6)
Desain Arsitektur Cisco Packet Tracer Mobile
Sifat Aplikasi Offline Client-Side (Ringan & Cepat)
Target Pengguna Siswa Kelas X SMAN 1 Belitang
